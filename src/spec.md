# Specification

## Summary
**Goal:** Remove all mountain decorative background elements and apply a solid dark grey background across all pages.

**Planned changes:**
- Remove mountain sunrise decorative background elements from the Layout component
- Update global CSS to remove background image declarations and apply solid dark grey background
- Remove CSS classes and JSX elements that reference mountain/sunrise decorative images
- Ensure consistent dark grey background across all pages (LandingPage, TasksPage, WalkTrackingPage, ExerciseCategoryPage, SettingsPage, TransformationPage, StatsPage, PartnerPage, LoginPage)

**User-visible outcome:** Users will see a clean, solid dark grey background across all pages instead of decorative mountain imagery.
